# Advanced React Todo App

A modern, feature-rich Todo application built using React.
This project focuses on clean UI, user experience, and real-world functionality.

## 🚀 Live Demo
[https://swift-todo-advanced.vercel.app/](https://swift-todo-advanced.vercel.app/)

## ✨ Features
- Add, edit, and delete todos
- Due date support
- Reminder toast notifications
- Dark mode with persistence
- Search and filter (All / Completed / Pending)
- LocalStorage data persistence
- Responsive and modern UI

## 🛠 Tech Stack
- React
- JavaScript (ES6+)
- CSS (Custom styling)
- Browser APIs (LocalStorage, Audio)
